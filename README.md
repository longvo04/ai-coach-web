# AI Coach FE

React + Vite frontend for an AI-powered career coach. Key features include CV analysis, goal planning and tracking, progress history, personalized feedback, and account management.

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run dev
   ```
3. Backend: ensure the API is available at `http://localhost:8080` (configurable in `src/api/axiosClient.js`).

Optional:
```bash
npm run lint
```
